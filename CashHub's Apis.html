CashHub API Documentation

1. User Credential APIs:

  • /signin - (POST) Requires Name, Number, Email, Password
  • /login - (POST) Requires Email and Password ## Gives you all the DATA of the particular User

2. User Account APIs:

  • /addaccount - (POST) Requires Account_number, Bank_name, IFSC_Code, Account_type, Balance, UPI
  • /getaccdetail/id - (GET) Gives you all account details of the particular User
  • /addhistory - (POST) Requires Reciever_name, Amount
  • /gethistory/id - (GET) Gives you all of the particular User's History
